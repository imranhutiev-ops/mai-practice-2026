Репозиторий содержит решения 7 ключевых задач по спортивному программированию, разобранных в рамках журнала учебной практики.

## Список задач

1. **Task1_Sortings** — Сортировки (Моделирование Bubble Sort)[cite: 1]
2. **Task2_FastTriangles** — Быстрые треугольники (Метод двух указателей)
3. **Task3_Grasshopper** — Непростой кузнечик (Динамическое программирование)
4. **Task4_Dispersion** — Дмитрий и дисперсия (Метод скользящего окна)
5. **Task5_Chorvody** — Хороводоводы (Комбинаторика и модульная арифметика)
6. **Task6_PathFinding** — Простой поиск путей (Обход в глубину — DFS)
7. **Task7_Dijkstra** — Алгоритм Дейкстры (Кратчайшие пути во взвешенных графах)

Все решения реализованы на C++ и оптимизированы под ограничения контестов по времени и памяти.
